## 📌 Sobre o projeto

O **Easy Shopping** foi um projeto desenvolvido com o objetivo de colocar em prática os conhecimentos adquiridos em aula, principalmente sobre:

- estruturação com HTML
- estilização com CSS
- adaptação para diferentes dispositivos
- responsividade para desktop e mobile

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

---

## 📱 Responsividade

Neste projeto, trabalhei a responsividade para que a interface ficasse bem ajustada em diferentes formatos de tela, oferecendo uma melhor experiência tanto no computador quanto no celular.
